## Dev Notes

**Monorepo Structure (from Architecture doc):**
```
maisum/
├── apps/
│   ├── mobile/              # Expo app
│   ├── admin-web/           # Next.js admin +um
│   └── restaurant-web/      # Next.js admin restaurante
├── packages/
│   ├── shared/              # Types TS, validacoes Zod, constantes
│   └── ui/                  # Componentes React (admin webs only)
├── supabase/                # Migrations, functions, seed
├── turbo.json
└── package.json
```

**Key decisions:**
- `packages/ui` serves ONLY admin webs (Next.js). Mobile tem componentes proprios (React Native).
- Design tokens compartilhados via Tailwind preset em `packages/shared/tailwind-preset.js`
- TypeScript strict mode em todo o monorepo
