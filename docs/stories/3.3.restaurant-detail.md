## Story
**As a** client user,
**I want** to see complete information about a restaurant,
**so that** I can decide whether to visit and use my coupon.

## Acceptance Criteria
1. Tela de detalhes com: carrossel de fotos, nome, nota media, numero de avaliacoes, tipo de cozinha (FR12)
2. Secao "Beneficio +um": lista dos itens disponiveis, horarios e dias em que o beneficio esta ativo, indicador de disponibilidade (FR12, FR21, FR22)
3. Secao "Sobre": descricao, endereco (tap abre Google Maps/Apple Maps), telefone (tap-to-call), horario de funcionamento (FR12)
4. Mini-mapa com localizacao e botao "Abrir no Maps" (deep link para Google Maps/Apple Maps) (FR12)
5. Secao de avaliacoes: ultimas 5 avaliacoes com nota, comentario e data (FR31)
6. Botao fixo no bottom: "Usar Cupom" (se tiver cupom disponivel), "Assinar para usar" (se nao assinante), "Sem cupons disponiveis" (se assinante sem cupons) (FR12, FR15)
7. Botao de chat: "Conversar com o restaurante" (FR26)
8. Indicador visual se beneficio esta disponivel agora (verde) ou fora do horario (cinza) (FR22)

## Dev Notes

**RPC Function (from Architecture doc):**
```sql
CREATE OR REPLACE FUNCTION get_restaurant_detail(p_restaurant_id UUID)
RETURNS JSON AS $$
  SELECT json_build_object(
    'restaurant', r,
    'avg_rating', COALESCE(
      (SELECT AVG(rating)::NUMERIC(2,1) FROM reviews WHERE restaurant_id = p_restaurant_id), 0
    ),
    'review_count', (SELECT COUNT(*) FROM reviews WHERE restaurant_id = p_restaurant_id),
    'benefits', (
      SELECT json_agg(b) FROM benefits b
      WHERE b.restaurant_id = p_restaurant_id AND b.is_active = true
    ),
    'rules', (
      SELECT json_agg(br) FROM benefit_rules br
      WHERE br.restaurant_id = p_restaurant_id AND br.is_active = true
    )
  )
  FROM restaurants r WHERE r.id = p_restaurant_id;
$$ LANGUAGE sql SECURITY DEFINER STABLE;
```

**Disponibilidade Calculation:**
```typescript
function isBenefitAvailableNow(rules: BenefitRule[]): boolean {
  const now = new Date()
  const currentDay = now.getDay() // 0=dom, 1=seg, ..., 6=sab
  const currentTime = `${now.getHours().toString().padStart(2, '0')}:${now.getMinutes().toString().padStart(2, '0')}`

  return rules.some(rule =>
    rule.is_active &&
    rule.available_days.includes(currentDay) &&
    currentTime >= rule.available_hours_start &&
    currentTime <= rule.available_hours_end
  )
}
```

**Deep Link para Maps:**
```typescript
import { Linking, Platform } from 'react-native'

function openInMaps(lat: number, lng: number, name: string) {
  const url = Platform.select({
    ios: `maps:0,0?q=${name}&ll=${lat},${lng}`,
    android: `geo:0,0?q=${lat},${lng}(${name})`,
  })
  if (url) Linking.openURL(url)
}
```
