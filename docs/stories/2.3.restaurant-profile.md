## Story
**As a** restaurant admin,
**I want** to manage my restaurant's profile,
**so that** clients see accurate and attractive information.

## Acceptance Criteria
1. Pagina de perfil no painel do restaurante com todos os campos editaveis
2. Upload/reordenacao de fotos (ate 5, drag-and-drop)
3. Edicao de: nome, descricao, endereco, telefone, horario de funcionamento
4. Preview de como o perfil aparece no app do cliente
5. Salvamento com botao de salvar e feedback visual
6. Validacao de campos obrigatorios

## Dev Notes

**Tabela `restaurants` — colunas editaveis pelo restaurant_admin:**
```sql
-- Colunas que o restaurant_admin pode editar via RLS
name TEXT NOT NULL,
description TEXT,
address TEXT NOT NULL,
phone TEXT,
cuisine_type TEXT,
latitude DOUBLE PRECISION NOT NULL,
longitude DOUBLE PRECISION NOT NULL,
photos TEXT[] DEFAULT '{}',
-- is_active NAO e editavel pelo restaurant_admin (apenas super_admin)
-- admin_user_id NAO e editavel pelo restaurant_admin
```

**RLS Policy (restaurant admin edita apenas o proprio restaurante):**
```sql
CREATE POLICY "Restaurant admin can update own" ON restaurants FOR UPDATE USING (admin_user_id = auth.uid());
CREATE POLICY "Restaurant admin can view own" ON restaurants FOR SELECT USING (admin_user_id = auth.uid());

-- Helper function usada em outras tabelas
CREATE OR REPLACE FUNCTION get_my_restaurant_id()
RETURNS UUID AS $$
  SELECT id FROM restaurants WHERE admin_user_id = auth.uid() LIMIT 1;
$$ LANGUAGE sql SECURITY DEFINER STABLE;
```

**Supabase Storage — upload de fotos pelo restaurant_admin:**
```sql
-- Policy: restaurant_admin pode fazer upload para sua pasta
CREATE POLICY "Restaurant admin upload photos" ON storage.objects FOR INSERT
  WITH CHECK (
    bucket_id = 'restaurant-photos'
    AND (storage.foldername(name))[1] = (SELECT id::text FROM restaurants WHERE admin_user_id = auth.uid() LIMIT 1)
  );

-- Policy: restaurant_admin pode deletar suas fotos
CREATE POLICY "Restaurant admin delete photos" ON storage.objects FOR DELETE
  USING (
    bucket_id = 'restaurant-photos'
    AND (storage.foldername(name))[1] = (SELECT id::text FROM restaurants WHERE admin_user_id = auth.uid() LIMIT 1)
  );

-- Policy: publico pode ler fotos
CREATE POLICY "Public read photos" ON storage.objects FOR SELECT USING (bucket_id = 'restaurant-photos');
```

**Zod Schema para validacao do perfil:**
```typescript
import { z } from 'zod'

export const restaurantProfileSchema = z.object({
  name: z.string().min(2, 'Nome obrigatorio (min 2 caracteres)'),
  description: z.string().optional(),
  address: z.string().min(5, 'Endereco obrigatorio (min 5 caracteres)'),
  phone: z.string().optional(),
  cuisine_type: z.string().optional(),
  latitude: z.number().min(-90).max(90),
  longitude: z.number().min(-180).max(180),
})
```
