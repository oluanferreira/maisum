## Story
**As a** client user,
**I want** to see my coupon history,
**so that** I know how many I have and where I used them.

## Acceptance Criteria
1. Tab "Cupons" no app mostra 3 secoes: Disponiveis, Usados, Expirados (FR14)
2. Cupons disponiveis mostram: quantidade restante, restaurantes ainda nao usados
3. Cupons usados mostram: restaurante, data de uso, item resgatado (FR20)
4. Cupons expirados mostram: quantidade que expirou no ultimo periodo (FR9)
5. Contador visual de cupons: "X de Y cupons restantes este mes/ano" (FR5)
6. Indicador de cupons extras (indicacao, avaliacao, post social) (FR32, FR34, FR40)
7. Barra de progresso visual do uso mensal

**Supabase queries for filtering by status:**
```typescript
// Available coupons count
const { count: available } = await supabase
  .from('coupons')
  .select('*', { count: 'exact', head: true })
  .eq('user_id', userId)
  .eq('status', 'available')

// Used coupons with restaurant info
const { data: usedCoupons } = await supabase
  .from('coupons')
  .select(`
    *,
    restaurant:restaurants(name, photos),
    benefit:benefits(name, category)
  `)
  .eq('user_id', userId)
  .eq('status', 'used')
  .order('used_at', { ascending: false })
  .range(offset, offset + limit)

// Expired coupons count for last period
const { count: expired } = await supabase
  .from('coupons')
  .select('*', { count: 'exact', head: true })
  .eq('user_id', userId)
  .eq('status', 'expired')
  .gte('expires_at', lastPeriodStart)

// Extra coupons by source
const { data: extras } = await supabase
  .from('coupons')
  .select('source')
  .eq('user_id', userId)
  .eq('status', 'available')
  .neq('source', 'subscription')
```
