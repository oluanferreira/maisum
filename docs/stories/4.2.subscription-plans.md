## Story
**As a** client user,
**I want** to choose a plan and pay easily,
**so that** I can start using my coupons.

## Acceptance Criteria
1. Tela de selecao de plano: card Mensal (R$19,90/mes, 10 cupons) e card Anual (R$89,90/ano, 100 cupons, badge "Melhor valor")
2. Ao selecionar plano, redireciona para checkout do AbacatePay (PIX, cartao, boleto)
3. Apos pagamento confirmado (webhook), app atualiza status automaticamente
4. Tela de sucesso com animacao e contagem de cupons recebidos
5. Se pagamento falhar, tela de erro com opcao de tentar novamente
6. Usuario sem assinatura ativa e redirecionado para tela de planos ao tentar usar funcionalidades premium

**AbacatePay Checkout Redirect Flow:**
```typescript
// apps/mobile/services/payments.ts
import * as WebBrowser from 'expo-web-browser'
import { supabase } from './supabase'

export async function createSubscription(planType: 'monthly' | 'annual') {
  // 1. Call Edge Function to create AbacatePay checkout session
  const { data, error } = await supabase.functions.invoke('create-checkout', {
    body: { plan_type: planType },
  })

  if (error) throw error

  // 2. Open AbacatePay checkout in in-app browser
  const result = await WebBrowser.openAuthSessionAsync(
    data.checkout_url,
    'maisum://payment-callback'  // Deep link return URL
  )

  return result
}
```

**Subscription Flow Sequence (from Architecture section 8.1):**
```
User (App) → Seleciona plano + paga
AbacatePay → Webhook: subscription.paid
Edge Function → UPDATE subscription status=active
Edge Function → INSERT payment
Edge Function → RPC allocate_coupons(10 ou 100)
Edge Function → Push "Pagamento confirmado!"
Push Notification → User recebe notificacao + cupons disponiveis
```

**Guard Hook Pattern:**
```typescript
// apps/mobile/hooks/useSubscriptionGuard.ts
import { useRouter } from 'expo-router'
import { useSubscriptionStore } from '@/stores/subscription'

export function useSubscriptionGuard() {
  const router = useRouter()
  const { isActive, isLoading } = useSubscriptionStore()

  const requireSubscription = () => {
    if (!isLoading && !isActive) {
      router.push('/plans')
      return false
    }
    return true
  }

  return { isActive, isLoading, requireSubscription }
}
```
