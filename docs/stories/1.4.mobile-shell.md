## Story
**As a** client user,
**I want** the mobile app to have a clear navigation structure,
**so that** I can navigate between core sections of the app.

## Acceptance Criteria
1. Expo Router configurado com file-based routing no `apps/mobile/app/`
2. Bottom tab navigation com 4 tabs: Home (House icon), Cupons (Ticket icon), Chat (ChatCircle icon), Perfil (User icon)
3. Stack navigation dentro de cada tab para sub-telas (restaurant detail, coupon detail, etc.)
4. Telas placeholder criadas para todas as core screens: Home, Cupons, Chat (lista + conversa), Perfil (index + subscription + referral), Restaurant detail, Coupon QR, Plans, Review
5. Splash screen configurada com logo placeholder
6. Safe area handling configurado para iOS (notch) e Android (status bar, gesture bar)
7. Supabase client inicializado e conectado no app com AsyncStorage para persistencia de sessao
8. Auth state listener configurado: redireciona para (auth)/login se nao autenticado, para (tabs) se autenticado
9. Tela de login/cadastro funcional com email/senha e botoes de login social (Google/Apple)

### Supabase Client (Mobile)

```typescript
// apps/mobile/services/supabase.ts
import { createClient } from '@supabase/supabase-js'
import AsyncStorage from '@react-native-async-storage/async-storage'
import { Database } from '@maisum/shared/types/supabase'

export const supabase = createClient<Database>(
  process.env.EXPO_PUBLIC_SUPABASE_URL!,
  process.env.EXPO_PUBLIC_SUPABASE_ANON_KEY!,
  {
    auth: {
      storage: AsyncStorage,
      autoRefreshToken: true,
      persistSession: true,
      detectSessionInUrl: false,
    },
  }
)
```

### Auth State Listener (Root Layout)

```typescript
// app/_layout.tsx (trecho do auth listener)
import { supabase } from '@/services/supabase'
import { useRouter, useSegments } from 'expo-router'

useEffect(() => {
  const { data: { subscription } } = supabase.auth.onAuthStateChange(
    (event, session) => {
      useAuthStore.getState().setSession(session)
      if (session) {
        // Fetch profile and redirect to tabs
        router.replace('/(tabs)')
      } else {
        router.replace('/(auth)/login')
      }
    }
  )
  return () => subscription.unsubscribe()
}, [])
```

### Tab Bar Config

```typescript
// app/(tabs)/_layout.tsx
import { Tabs } from 'expo-router'
import { House, Ticket, ChatCircle, User } from 'phosphor-react-native'

export default function TabLayout() {
  return (
    <Tabs
      screenOptions={{
        tabBarActiveTintColor: '#FF6B35',    // --color-primary
        tabBarInactiveTintColor: '#9CA3AF',  // --color-neutral-400
        tabBarStyle: { height: 56, paddingBottom: 8 },
        headerShown: false,
      }}
    >
      <Tabs.Screen name="index" options={{
        title: 'Home',
        tabBarIcon: ({ color, size }) => <House size={size} color={color} />,
      }} />
      <Tabs.Screen name="coupons" options={{
        title: 'Cupons',
        tabBarIcon: ({ color, size }) => <Ticket size={size} color={color} />,
      }} />
      <Tabs.Screen name="chat" options={{
        title: 'Chat',
        tabBarIcon: ({ color, size }) => <ChatCircle size={size} color={color} />,
      }} />
      <Tabs.Screen name="profile" options={{
        title: 'Perfil',
        tabBarIcon: ({ color, size }) => <User size={size} color={color} />,
      }} />
    </Tabs>
  )
}
```
