## Story
**As a** client user,
**I want** to invite friends and earn extra coupons,
**so that** I get rewarded for spreading the word.

## Acceptance Criteria
1. Tabela `referrals` criada: id, referrer_id, referred_id, status (pending | completed), bonus_granted, created_at (FR33)
2. Tela "Indicar Amigos" no perfil com link de indicacao unico (FR33)
3. Link usa deep linking (Expo) para abrir app ou app store (FR33)
4. Quando indicado se cadastra e assina, ambos recebem 3 cupons extras (FR34)
5. Status muda de pending para completed apos assinatura do indicado (FR34)
6. Cupons extras respeitam limite de 10 extras/mes (FR35)
7. Historico de indicacoes: quantos indicados, quantos convertidos, cupons ganhos
8. Share sheet nativo para compartilhar link via WhatsApp, Instagram, etc.

## Dev Notes

**Referrals Table DDL (from Architecture doc section 5.1):**
```sql
CREATE TABLE referrals (
  id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
  referrer_id UUID NOT NULL REFERENCES profiles(id),
  referred_id UUID REFERENCES profiles(id),
  status referral_status DEFAULT 'pending',
  bonus_granted BOOLEAN DEFAULT false,
  created_at TIMESTAMPTZ DEFAULT now()
);

CREATE INDEX idx_referrals_referrer ON referrals(referrer_id);
CREATE INDEX idx_referrals_referred ON referrals(referred_id);
```

**Referral code in profiles (from Architecture doc):**
```sql
-- profiles table includes:
referral_code TEXT UNIQUE DEFAULT encode(gen_random_bytes(6), 'hex')
```

**Referral Flow (from Architecture doc section 8.4):**
```
Referrer → Gera link (referral_code) → Compartilha (WhatsApp, etc.)
Referred → Abre app via deep link → Cadastro com referral_code → Assina plano
AbacatePay → Webhook subscription.paid → Edge Function
Edge Function → UPDATE referral status=completed
             → grant_extra_coupons(referrer, 3, 'referral')
             → grant_extra_coupons(referred, 3, 'referral')
             → Push "Seu amigo assinou! +3 cupons"
```

**Referral Screen (from Frontend Spec 4.1.11):**
- Titulo "Indica e Ganha!" com descricao do bonus
- Card com link copiavel: `maisumapp.com/ref/{code}`
- Botao copiar + botoes de share (WhatsApp, generico)
- Historico abaixo: nome do indicado, status (Pendente/Convertido), cupons ganhos

**Deep Linking (Expo Router):**

- Expo Router captura parametro `ref` na tela de cadastro
