## Story
**As a** client user,
**I want** to see nearby partner restaurants on a map,
**so that** I can discover places to use my "+um" benefit.

## Acceptance Criteria
1. Home screen com mapa interativo ocupando ~50% da tela (React Native Maps + Google Maps) (FR10)
2. Markers no mapa para cada restaurante ativo na cidade do usuario (FR10)
3. Mapa centralizado na localizacao atual do usuario (com permissao de localizacao) (FR10)
4. Se localizacao negada, centraliza na cidade selecionada (FR58)
5. Tap no marker mostra preview card do restaurante (nome, foto, tipo, distancia) (FR10)
6. Tap no preview card navega para detalhes do restaurante (FR12)
7. Lista de restaurantes scrollavel abaixo do mapa (cards) (FR10)
8. Pull-to-refresh atualiza dados (NFR1)

## Dev Notes

**RPC Function (from Architecture doc):**
```sql
-- Get nearby restaurants (simple distance calc)
CREATE OR REPLACE FUNCTION get_nearby_restaurants(
  p_lat DOUBLE PRECISION,
  p_lng DOUBLE PRECISION,
  p_radius_km DOUBLE PRECISION DEFAULT 10
)
RETURNS SETOF restaurants AS $$
  SELECT *
  FROM restaurants
  WHERE is_active = true
    AND (
      6371 * acos(
        cos(radians(p_lat)) * cos(radians(latitude)) *
        cos(radians(longitude) - radians(p_lng)) +
        sin(radians(p_lat)) * sin(radians(latitude))
      )
    ) <= p_radius_km
  ORDER BY (
    6371 * acos(
      cos(radians(p_lat)) * cos(radians(latitude)) *
      cos(radians(longitude) - radians(p_lng)) +
      sin(radians(p_lat)) * sin(radians(latitude))
    )
  );
$$ LANGUAGE sql SECURITY DEFINER STABLE;
```

**RestaurantCard Molecule (from Frontend Spec):**
```
┌────────────────────────────────────┐
│ [Foto 16:9]                        │
├────────────────────────────────────┤
│ Nome do Restaurante          ★ 4.5 │
│ Italiana · 1.2 km                  │
│ ┌──────────────────────┐           │
│ │ 🎁 Pizza gratis      │           │
│ └──────────────────────┘           │
│         [Disponivel agora] 🟢      │
└────────────────────────────────────┘
```
- Width: 100% (lista) ou 280px (horizontal scroll)
- Foto: aspect ratio 16:9, radius top `--radius-lg`
- Padding interno: `--space-4` (16px)
- Shadow: `--shadow-sm`, hover: `--shadow-md`
- Badge de disponibilidade: verde (agora) ou cinza (fora do horario)
- Tag de beneficio: `--color-primary-light` bg, `--color-primary-dark` text
